# MediQueue

MediQueue is an online tutor booking platform where students can find tutors, book sessions, and manage their learning journey easily.

## 🌐 Live Site
https://medi-queue-client-side.vercel.app/

## 🚀 Features

- 🔍 Search tutors by language, subject, or expertise
- 📚 View detailed tutor information and available session slots
- 📝 Book tutoring sessions securely after login
- 👨‍🏫 Add and manage your own tutor services
- 📅 Track all booked tutoring sessions from dashboard
- 🌙 Dark and light theme toggle support
- 🔐 JWT authentication with protected routes
- 📱 Fully responsive design for mobile, tablet, and desktop

## 🛠️ Technologies Used

- Next.js
- React
- Tailwind CSS
- MongoDB
- Express.js
- JWT Authentication
